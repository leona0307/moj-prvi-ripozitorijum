- репозиторијум је место где чувамо код. Он може бити приватан и јаван. Приватан репо може да види само власник и овлашћена лица. Јавни репо може да виде свако.

- да бисмо иницијализовали рипозиторујум, ако већ није иницијализован, покрећемо команду:
    - git init

- додавање нових фалова, садржаја или измене у постојећим фајловима, вршимо командом:
    - git add [ime fajla] 
- ако желимо да пошаљемо у репо све измене онда користимо:
    - git add .
